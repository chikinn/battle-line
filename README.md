# Battle Line
#### Robert B. Kaspar (rbkaspar@gmail.com) and Ben Zax

Computer players for a poker-inspired two-player card game

Intended to run in Python 3 (change the shebang in `bl_wrapper.py`)

## Example usage
    $ ./bl_wrapper.py racist naive
